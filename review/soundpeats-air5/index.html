大音量の音声ガイダンスにさようなら Soundpeats Air5 レビュー

2024/09/25
Tags: Device

サムネ画像

その人のライフスタイルやら思想やらが反映されるというオーディオ機器選び。皆さまの崇拝するサウンド環境はどのようなものでしょうか。因みに私はインナーイヤー教無線派、コスパ重視主義に属しています。

  • インナーイヤー → 自室で長時間使用するため
  • 無線 → 動画を流しながら運動したいため
  • コスパ重視 → 使用時間が長くてバッテリー劣化が早く、買い替えスパンが短いため

上記に該当する商品としてこれまで『Anker Soundcore Life Note 3S』を使用していたのですが、購入から1年半が経過した頃にある事実が発覚しました。

バッテリー残量低下時の警告音がクソデカいのです。

唐突に耳元で炸裂する「Battery Low!!!」のアナウンス。これはバッテリー残量が30%、20%、10％を下回る度に発動するようで、おかげでバッテリーが劣化した私のイヤホンは悲しきビックリ量産マシーンになってしまったのです。接続時は「ピポパ♪」という控えめで可愛らしい電子音だったのに、なぜ急にネイティブの心を宿してしまったのか。

かくして、私のオーディオ宗教に「音声ガイダンスを調整できること」という新たな戒律が刻まれ、新たなイヤホン探しの旅が始まりました。

そして条件を満たすイヤホンを探し回り、最終的に辿りついたのが『SOUNDPEATS Air5』というイヤホンです。果たして、事前情報通り音声ガイダンスを消すことは出来るのか…？

※余談ですが、Life Note 3Sの後継機である『Anker Soundcore K20i』について、メーカーのチャットサポートに問い合わせたところ、「K20iに音声ガイダンスの調整機能はなく、またその音量も前機種と同等程度である」という返答を頂いてしまったので、今回は見送りとなりました。（2024年9月中旬時点）

使用環境

  • 使用場所：自室（そこそこ静か）
  • デバイス：デスクトップPC＋iPad（アプリで設定する用）
  • 用途：動画やゲーム中心
  • 使用時間：12時間ほど？

上記の環境で使用した上でのレビューとなります。

外観

まずは商品内容と外観について確認していきます。

画像：商品の外箱と内容物をまとめておいた写真

商品内容はイヤホン＆ケース本体、Type-C充電ケーブル、取扱説明書、謎マスコットのステッカー。ケーブルは土台の裏側に隠れており長さは最低限となっています。まぁTypc-C充電ケーブル持ってないぜって人は殆どいないので問題ないでしょう。また、取扱説明書は日本語も記載されています。

画像：ケースを開けた画像

色はホワイトを選びましたが…ケースとイヤホンで色が違うのがちょっと気になる点ですね。商品画像だとホワイト一色で統一されてそうなイメージになっていますが、実際にはケースは青みがかったグレーになっており、ケースの中でもやや白い部分とグレーな部分で差異があります。せめてケースの中だけでも色味を統一して欲しかった感。

ケースは丸みを帯びた形で、イヤホンを縦に挿入して収納するタイプです。

個人的には机に置いて使うので平置きして充電するタイプの方が使い勝手がいいですが…ここは好みでしょう。ただ、蓋側が結構深いため、そこにイヤホンが入り込んで取り出しにくいのは微妙なところ。

画像：ランプが点灯している状態

点灯部分は一か所のみ。色や明滅でバッテリー残量を把握する仕組みになっています。複数ランプをついてる方が直感的で好きですが、実用的にはこちらは問題なさそうです。赤けりゃとりあえず充電しておけばいいのよ！

画像：イヤホンを複数並べた写真

イヤホンの大きさはこんな感じです。左は先代イヤホン、右は初代AirPods君です。先代と並ぶと小さく見えますが、一般的なコンパクトなサイズのイヤホンと考えてよさそうです。

アプリの機能

本記事の主題ともいえる部分です。果たして音声ガイダンスを調整することは出来るのか。

イヤホンの設定にはSOUNDPEATSが提供する『PeatsAudio』というアプリを導入する必要があります。その前身である『SOUNDPEATS』というアプリを入れても無駄足になるので注意しましょう。（一敗）

このアプリ、利用するには会員登録をする必要があり、またアプリの評価もすこぶる悪くてちょっと不安になりますが、買ってしまったものは仕方ない…！

画像：SoundPeatsの設定画面

会員登録を済ませ、イヤホンをペアリングして設定画面を開きました。早速、音声ガイダンスに関する項目を探します。

画像：音声ガイダンスに関する項目

音声ガイダンスの調整機能が…ある！

事前の情報通り、アナウンスの音量を調整できる項目がありました。早速限界まで下げました。ガイダンス消滅！

いやー…最初にケースの蓋を開けた際「Pairing!!（お風呂場反響ボイス）」などと高らかに流れていたので…これで消せなかったらどうしようかと思っていました。本当に消せてよかった。調整できてありがとう。

この後、念のためバッテリーが0%になるまで使用しましたが、特に警告音が流れることも無く、静かに力尽きていました。

他にもイヤホンタップ時の挙動やノイズキャンセリングやゲームモードの切り替え、イコライザーなどのカスタマイズ設定が可能です。タップ時の操作は誤爆しやすい機能でもあるため、ちゃんとオフに出来るのは助かりますね。（先代のSoundcore君、なぜかオフに出来なかったのよね…）

なお、初回はファームウェアのアップデートをお忘れなく。一度アップデートせずにPCで使用したのですが、1000円イヤホン並の音質になっていました。

音質

次に、イヤホンの核ともいえる音質について。

ただ予めお断りしておきますと、私は音楽や音質について知識もこだわりもなく、音の良し悪しも分かりません。音の評価はいつだって前のイヤホンとの相対評価！そういった人間が書いた感想であることをご了承ください。

さて、開き直って先代イヤホンのAnker Soundcore Life Note 3Sと比較しますと、音がはっきりと聞こえるように思います。

重低音もよく出ていて、普段はあまり聞かない音楽を流してみますと音の違いをちゃんと感じ取ることが出来ました。個々の楽器の音色が分かりやすく、臨場感もあるので音楽を聴くのはこちらの方が楽しめそうですね。

一方で、そのはっきりとした音が気になる部分もありました。

顕著だったのが普段から遊んでいるアクションゲームで、打撃音や金属音がキンキンとした音で耳障りに感じました。また動画では人の声やゆっくりボイスに電子音っぽさが目立つなど、私の用途とは今一つ噛みあっていない印象です。

これらはプリセットのイコライザーでは解決せず、カスタムイコライザーで気になる音を下げることで緩和することは出来ました。私の耳においては、高音域をしっかり目に下げ、重低音も少し下げることで比較的聞きやすい音になったと思います。

その他

バッテリー

公式に仕様によると最大再生時間は6時間、ケース併用で約30時間とのこと。

音声ガイダンスの確認も兼ねてバッテリーが0%になるまで使用したところ、確かに満充電から6時間程度でバッテリーが切れました。ただし無音の時間もそれなりにあったはずなので、音を流し続けた場合はもう少し早く充電が切れるような気がします。

また、因果関係は定かではありませんが、バッテリー残量が10%を切ったあたりから時々音が途切れる現象も発生していました。

ノイズキャンセリング

私は使わない機能ですが、ノイズキャンセリングも搭載しています。

実際に試したところ、PCやエアコン（風量控えめ）の駆動音くらいなら概ね違和感なく無音にしてくれます。一方でエアコンでも風量が上がって音が大きくなると音が多少貫通しました。

また、家の近くでタイミングよく洗浄作業があったので使用してみたところ「ﾄﾞﾄﾞﾄﾞﾄﾞﾄﾞ…」という音が「ｺｺｺｺｺ…」くらいに軽減されたものの、イヤホンからは常時「ｽｰｰｰｰｯ」という、必死に音を打ち消している様子が漏れ出ていました。まぁ流石にね…

元がインナーイヤーですのでヘッドホンのような静寂の世界…！とはいきませんが、周囲のノイズを軽減して音楽を聴きたい、くらいの用途には十分使えるのではないでしょうか。

マルチポイント接続

購入前は特に意識してなかったのですが、2台のデバイスに同時接続できるマルチポイント接続機能も搭載されています。

これが結構便利で、イコライザーの設定時にはPCの音を聞きながら調整できたので非常に助かりました。先代イヤホンでは設定変更の度にiPadと接続を切り替える必要がありましたからね…デバイス間の接続も1~2秒で勝手に切り替わりますし、あって損のない機能だと思います。

総評

「インナーイヤーで音楽を楽しみたい人には良さそう。何より音声ガイダンスの調整機能がエラい！」です。

機能面で充実しており、音声ガイダンスも消せるのが非常に嬉しい。一方で個人的には高音の強さやケースの色のちぐはぐさ、イヤホンの出し入れのしにくさなど少し気になる点もありました。

音についてはカスタムイコライザーを使うことで実用可能な範疇に収まりましたが、「気に入った！次も同じものを買う！」とまではいかず。他に似たスペックのイヤホンが見つかれば、次はそちらを試したくなるのだろうなと思います。

まだ使用時間が短く、今回評価できなかった通信の安定性やバッテリーの耐久性については後日追記するかもしれません。以上、SOUNDPEATS Air5のレビューでした。

…余談ですが、SOUNDPEATSも中国メーカーによくある頻繁なセールやクーポン配布を行っています。私の場合、トータルで約2000円引きの7700円程度で購入できました。購入時の参考にドウゾ。
